# Task Manager - Internship Project

## Overview
This is a Task Management System built as an internship project.  
It allows users to **add, edit, delete, and manage tasks** with a professional, modern interface.  

- Frontend: React.js + Tailwind CSS + React Icons  
- Backend: Node.js + Express.js  
- Database: MongoDB  

---

## Features
- Add a new task  
- Edit task title inline  
- Mark tasks as completed / pending  
- Delete tasks  
- UI with hover effects and icons  

Optional advanced features like authentication, filters, and progress tracking can be added.
